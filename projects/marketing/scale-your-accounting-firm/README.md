# Scale Your Accounting Firm — course notes

> **Status:** Active · **Owner:** Julia · **Started:** 2026-07

Digested notes from the "Scale Your Accounting Firm" advisory program, organized
by track and module, feeding into JK Accounting Group's marketing strategy.

## Purpose

Julia (and Lilian) are taking this course as a program to grow the firm. Rather
than storing full video transcripts, this project keeps a **digested version**
of each video — the core concepts plus, where the video calls for it, concrete
next steps for JK's own marketing. The point is to build a running body of
knowledge we can act on, not an archive of raw transcripts.

## What's here

Each track gets its own folder; each module/stage within a track gets a
subfolder; each video gets one numbered digest file inside that subfolder.
Add tracks/modules the same way as Julia and Lilian progress through the
course.

data). Actionable items that turn into real marketing work get built out in
the relevant project (e.g. `referral-offer-strategy/`, `consultation-booking/`)
rather than living here — this project is the source of the ideas, not where
they get executed.
