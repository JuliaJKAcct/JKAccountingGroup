# Starter prompts

Copy/paste these into the Claude Project to generate common collateral. Replace
the **[bracketed]** parts. Everything stays on-brand because the Project already
carries the brand guide.

## Business card
```
Design a business card for [NAME], [TITLE] at JK Accounting Group.
Front: logo + firm name. Back: name, title, phone [PHONE], email [EMAIL],
[Miami / Fort Lauderdale / online], website. US size (3.5×2in, 300dpi, with
bleed). Bilingual line in EN + RU. Calm, premium, lots of ivory space.
Give me the layout and the print-ready spec.
```

## Instagram square post
```
Create a 1080×1080 Instagram post: topic "[e.g. 3 things foreign owners get
wrong on their first US tax return]". Mono uppercase kicker, Source Serif 4
headline, short IBM Plex Sans supporting line. Petrol-teal background, ivory
text, one small bronze accent. No fear-mongering. Then give me an EN caption and
an RU caption (native, not translated), plus 5 hashtags.
```

## Instagram carousel
```
Plan a 5-slide IG carousel (1080×1350) explaining "[topic]" to Ukrainian/Russian-
speaking US business owners. Slide 1 hook, slides 2–4 one point each, slide 5 a
calm CTA to book a free consultation. For each slide: kicker, headline, body, and
which brand color/background. Keep the voice reassuring, plain, bicultural.
```

## Story / reel cover
```
Design a 1080×1920 story cover for "[announcement]". Deep-teal background, ivory
serif headline, mono kicker, emblem logo top-left (not boxed), one bronze accent.
```

## Quote / testimonial graphic
```
Make a 1080×1080 testimonial graphic. Quote: "[REAL verified quote]" — [first
name, city]. Only use this exact quote; do not invent or embellish. Ivory card,
teal type, mono attribution line, small monogram.
```

## Flyer / one-pager
```
Design a one-page US-Letter flyer for "[service]" aimed at [audience]. Sections:
kicker headline, 3 plain-language benefits, who it's for, one CTA to book a free
consultation. Brand colors and type only. Real facts only — flag anything you'd
need me to confirm. 300dpi print-ready.
```

## Email / letterhead header
```
Create a letterhead header for JK Accounting Group: horizontal lock-up logo,
contact line, thin bronze rule. US Letter width. Then a matching plain-text email
signature for [NAME], [TITLE].
```

## On-brand check (run on anything)
```
Review the attached/last asset against JK-Brand-Guide.md §9 checklist. List any
off-palette color, wrong/boxed logo usage, non-brand font, invented fact, or
off-voice copy, and give me a corrected version.
```
